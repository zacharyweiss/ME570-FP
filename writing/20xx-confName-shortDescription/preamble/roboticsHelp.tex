This file introduces commands for drawing axes and cameras.

\begin{itemize}
\item \var{ax[label][opts]\{refid\}\{x\}\{y\}\{rot\}} Draws a pair of 2-D axes. \var{refid} can be the used to put labels and third axes.
\item \var{axup[label_rotation]\{refid\}\{label\}\{rot\}}
\item \var{axlabels\{refid\}\{label1\}\{label2\}}
\item \var{camera[label]\{refid\}\{x\}\{y\}\{rot\}}
\item \var{gpoint\{coordinates\}\{label\}\{rot\}}
\end{itemize}
